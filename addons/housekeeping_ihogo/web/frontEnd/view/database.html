分类表 ihogo_wdd_category

字段名 对应功能 备注
name 分类名称
id 分类唯一标识

评论表 ihogo_wdd_comment

字段名 对应功能 备注
Title 所评论内容的分类
content 评论的内容
avatar 用户头像

公司表 ihogo_wdd_company

字段名 对应功能 备注
pinyin 公司名称拼音 用于在小程序中按首字母检索,必须是字母
post_list 该公司发布的Post文章 多个用||分隔
name 公司名称

文章表 ihogo_wdd_post

字段名 对应功能 备注
Title 文章标题
id post文章唯一标识
FeaImage 文章缩略图地址 图片上传到腾讯云/七牛云等云存储后获得的图片外链地址
category 文章所属分类

问题表 ihogo_wdd_question

字段名 对应功能 备注
time 提问时间
question 提问内容
openid 小程序用户唯一标识
name 用户名
avatar 用户头像地址

轮播图表 ihogo_wdd_slideimg

字段名 对应功能 备注
imgurl 轮播图地址

用户表 ihogo_wdd_user

字段名 对应功能 备注
id 小程序用户openid
fav_post 用户收藏的问题
